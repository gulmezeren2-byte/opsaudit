# Security Policy

## Supported versions

| Version | Supported |
|---------|-----------|
| 0.1.x   | ✓ |

## Reporting a vulnerability

Please report vulnerabilities privately via [GitHub Security Advisories](https://github.com/gulmezeren2-byte/opsaudit/security/advisories/new) — do not open a public issue for security problems.

`opsaudit` is designed with a minimal surface: it reads local CSV files, writes nothing, makes no network calls, and has two runtime dependencies (pandas, numpy). Reports about anything violating those properties are especially welcome.
